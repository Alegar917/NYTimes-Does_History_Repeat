Data

The dataset that I used was from The New York Times's The Archive API, an API that returns an array of NYT articles for a given month, going back to 1851. You simply pass the API the year and month and it returns a JSON object with all articles for that month.

Example Call

https://api.nytimes.com/svc/archive/v1/{year}/{month}.json?api-key={yourkey}
